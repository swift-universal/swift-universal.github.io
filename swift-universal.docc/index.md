Building for the everywhere computer.

## Overview

Swift Universal provides cross-platform libraries and tools that power the Apple ecosystem and beyond. We focus on shared foundations, robust subprocess management, and abstract interfaces.

## Topics

### Libraries

- **swift-figlet-kit**: ASCII art generation for Swift CLIs.
- **any-language-model**: Abstract interfaces for LLM integration.
- **common-process**: Robust subprocess management.
